# MyCV - Portfolio Project

این مخزن شامل کد منبع رزومه شخصی و پورتفولیو من است که با تمرکز بر تخصص‌های هوش مصنوعی و توسعه وب طراحی شده است.

## تکنولوژی‌های استفاده شده
* **Frontend:** HTML5, CSS3 (Modern Glassmorphism), JavaScript
* **Fonts & Icons:** Vazirmatn, FontAwesome 6.4
* **Deployment:** GitHub Pages

## پروژه‌های شاخص معرفی شده
1.  **Cybersecurity & NLP:** تشخیص حملات سایبری با مدل‌های BERT و غلبه بر چالش داده‌های نامتوازن (Imbalanced Data) با استفاده از Focal Loss.
2.  **fMRI Signal Processing:** تحلیل داده‌های مغزی با استفاده از شبکه‌های عصبی اسپایکی (SNN) و خوشه‌بندی معنایی لیبل‌ها توسط LLM.
3.  **Full-Stack Development:** توسعه پنل مدیریتی با React، Refine و Django.

## نحوه اجرا
برای مشاهده این رزومه، کافی است فایل `index.html` را در مرورگر خود باز کنید یا از لینک مستقیم GitHub Pages (در صورت فعال‌سازی) استفاده نمایید.
